weechat-eventc

An eventd client plugin

weechat-eventc provides events dispatching for WeeChat to eventd.

Dependencies

  • WeeChat
  • libeventc (distributed with eventd)

Git repository

The repository is hosted on GitHub: weechat-eventc Git repository

You can clone the project with Git by running:
$ git clone git://github.com/sardemff7/weechat-eventc
